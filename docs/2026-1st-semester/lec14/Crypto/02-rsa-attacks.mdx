# RSA 暗号への攻撃手法

今回は、CTF や実際のシステムでよく見られる2つの代表的な攻撃手法を解説します．

## 1. $N$ が小さい RSA 暗号

現代の安全な RSA 暗号では $N$ に 2048 ビット（約 617 桁）以上の巨大な数が使われますが、もし $N$ のビット数が小さい場合、すぐに素因数分解ができてしまいます．

素因数分解を行うアルゴリズムは主に

- 試し割り法
- pollard's rho 法
- 楕円曲線法
- 一般数体篩法

などがあげられます．

今回は，これらのアルゴリズムは扱わず，python のライブラリに頼る方向で行きます．

`sympy` の `factorint` 関数を使うことで、比較的小さな整数であればローカル環境で素因数分解を実行できます．

`sympy` は pycryptodome と同様に外部パッケージなので，インストールする必要があります．

<details>

<summary>SymPy のインストール</summary>

[pycryptodome](../../lec09/Crypto/01-environment.mdx#必要なライブラリのインストール) と同様に

`pip install sympy`

を実行してください．

```python
from sympy import factorint

factors = factorint(2026)
print(factors) 
```

```shell
{2: 1, 1013: 1}
```

素因数分解の結果が辞書型で返されます．$2026 = 2^{1} \cdot 1013^{1}$ であることがわかります．

</details>

素因数 $p, q$ が手に入れば、通常の RSA 暗号の手順通り $\varphi(N) = (p-1)(q-1)$ と $d \equiv e^{-1} \bmod{\varphi(N)}$ を計算して，復号が行えます．

## 2. $e$ が小さすぎる RSA 暗号 Low Public Exponent Attack

RSA 暗号の $e$ には、計算速度を高速化するために通常 $e = 65537$ （ $2^{16} + 1$ ）がよく用いられます．\
二進数で表すと，`10000000000000001` と最上位桁と最下位桁以外が `0` となり色々都合が良い数字です．\
ちなみに 16進数で表すと `10001` です．

CTF では，この $e$ を極端に小さい数にして出題することがあります．

ここで、平文メッセージ $m$ のサイズが小さく、公開鍵 $N$ が非常に大きい場合、以下の状態が発生することがあります．

$$
m^e < N
$$

### なにが問題であるのか？

RSA 暗号の暗号化は

$$
c = m^{e} \bmod{N}
$$

であったことを思い出すと，$m^{e} < N$ なので，$c = m^{e}$ です．
($m^{e} > N$ であれば，$c = m^{e} - N \cdot a$ です．)

剰余演算 $\bmod{N}$ が**一回も行われません**．
つまり、あまりの計算が発生しないため、$c$ の $e$ 乗根 $\sqrt[e]{c}$ を求めるだけで，$m$ が手に入ります．

こうなると、公開鍵 $N$ も秘密鍵 $d$ も素因数分解も一切関係なくなります．

累乗根を求めるためのアルゴリズムはニュートン法などがありますが，素因数分解と同様に外部パッケージに頼ります．

`gmpy2` の `iroot` 関数を用いることで，累乗根を求めることができます．

`gmpy2` も `sympy` や `pycryptodome` と同様にインストールする必要があります．

<details>

<summary>gmpy2 のインストール</summary>

`pip install gmpy2`

を実行してください．

```python
from gmpy2 import iroot

res = iroot(27, 3)
print(res)
```

```shell
$ python test.py
(mpz(3), True)
```

$27$ の $3$ 乗根を計算し，底は $3$ であり，$27$ は完全に $3^{3}$ と一致するということがわかります．

ちなみに，$27$ の $4$ 乗根を計算してみると，

```shell
(mpz(2), False)
```

が得られます．$27 \ne 2^{4}$ なので完全一致ではなく，`False` と出力されているのがわかります．この $2$ は近似値であり，$n^{4}$ の中で，$27$ に一番近くなるような $n$ が選択されます．

</details>
