## 今回学ぶこと

### 1. 計算量の考え方

入力サイズ `N` に対して、処理がだいたい何回行われるかを表します。

```c
// O(N): 一重ループ
for (int i = 0; i < n; i++) {
    sum += a[i];
}

// O(N^2): 二重ループ
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        // ...
    }
}
```

### 2. 基本ソート

配列を昇順（または降順）に並べ替える基本アルゴリズムです。

```c
// バブルソート（隣接要素を比較して交換）
for (int i = 0; i < n - 1; i++) {
    for (int j = n - 1; j > i; j--) {
        if (a[j - 1] > a[j]) {
            int t = a[j - 1]; a[j - 1] = a[j]; a[j] = t;
        }
    }
}
```

### 3. 線形探索と二分探索

```c
// 線形探索: O(N)
for (int i = 0; i < n; i++) {
    if (a[i] == key) return i;
}

// 二分探索（ソート済み配列で使う）: O(log N)
int lo = 0, hi = n - 1;
while (lo <= hi) {
    int mid = (lo + hi) / 2;
    if (a[mid] == key) return mid;
    else if (a[mid] < key) lo = mid + 1;
    else hi = mid - 1;
}
```
