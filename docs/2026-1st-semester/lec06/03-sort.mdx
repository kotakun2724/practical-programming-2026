# 基本ソート

配列を昇順や降順に並べ替える操作を**ソート**といいます。
ここでは代表的な2つの基本ソート、**挿入ソート**と**バブルソート**を学びます。

## 挿入ソート

### 考え方

トランプを左から1枚ずつめくって、すでに手札の中で正しい位置に挿入していくイメージです。
**左側の `i` 個はすでに整列済み**という不変条件を保ちながら進めます。

```
[3] 1 4 1 5      ← i=1: 3 は整列済み
[1 3] 4 1 5      ← i=2: 1 を挿入
[1 3 4] 1 5      ← i=3: 4 を挿入
[1 1 3 4] 5      ← i=4: 1 を挿入
[1 1 3 4 5]      ← 完成
```

### 実装

```c
#include <stdio.h>

void insertion_sort(int *a, int n) {
    for (int i = 1; i < n; i++) {
        int v = a[i];
        int j = i - 1;
        while (j >= 0 && a[j] > v) {
            a[j + 1] = a[j];
            j--;
        }
        a[j + 1] = v;
    }
}

int main(void) {
    int a[] = {5, 2, 4, 6, 1, 3};
    int n = 6;
    insertion_sort(a, n);
    for (int i = 0; i < n; i++) printf("%d ", a[i]);
    printf("\n");
    return 0;
}
```

### 計算量

| ケース | 計算量 |
|:-------|:-------|
| 最良（すでに整列済み） | `O(N)` |
| 平均 | `O(N^2)` |
| 最悪（逆順） | `O(N^2)` |

:::note 安定ソート
等しい要素どうしの順序が保たれるソートを**安定ソート**といいます。
挿入ソートは安定ソートです。
:::

## バブルソート

### 考え方

隣り合う2要素を比較して、逆順なら交換する。これを最後まで繰り返すと、
**最も小さい値が右端から左端へ「泡のように」浮かび上がってきます**。

### 実装（AOJ ALDS1_2_A の典型）

`a` を昇順に並べ替える、よく使われる書き方です:

```c
#include <stdio.h>

int bubble_sort(int *a, int n) {
    int swap_count = 0;
    for (int i = 0; i < n - 1; i++) {
        for (int j = n - 1; j > i; j--) {
            if (a[j - 1] > a[j]) {
                int t = a[j - 1];
                a[j - 1] = a[j];
                a[j] = t;
                swap_count++;
            }
        }
    }
    return swap_count;
}

int main(void) {
    int a[] = {5, 3, 2, 4, 1};
    int n = 5;
    int cnt = bubble_sort(a, n);
    for (int i = 0; i < n; i++) printf("%d ", a[i]);
    printf("\nswap = %d\n", cnt);
    return 0;
}
```

### 計算量

| ケース | 計算量 |
|:-------|:-------|
| 最良（すでに整列済み） | `O(N)`（早期終了を入れた場合） |
| 平均 | `O(N^2)` |
| 最悪（逆順） | `O(N^2)` |

バブルソートも安定ソートです。

:::warning swap 回数のカウントに注意
AOJ `ALDS1_2_A` は **交換回数（swap count）** の出力を求めます。
`if` の中で `swap` を実行したときだけインクリメントすることに注意しましょう。
:::

## 挿入ソート vs バブルソート

| 項目 | 挿入ソート | バブルソート |
|:-----|:-----------|:-------------|
| 最良 | `O(N)` | `O(N)`（早期終了） |
| 平均 | `O(N^2)` | `O(N^2)` |
| 最悪 | `O(N^2)` | `O(N^2)` |
| 安定性 | あり | あり |
| 実装のしやすさ | 普通 | やさしい |
| 実際の速度 | バブルより速いことが多い | 遅め |

:::tip 大量データには使わない
基本ソートは実装が簡単な反面、`N` が大きくなると遅すぎます。
`N` が数万を超える場合は、次回学ぶ **マージソート**（`O(N log N)`）や、
C++ なら `std::sort` を使いましょう。
:::
