### 各問題のヒント

#### ALDS1_1_A: Insertion Sort

外側ループで挿入位置を進め、内側ループで左へ比較・交換します。

```c
for (int i = 1; i < n; i++) {
    int v = a[i];
    int j = i - 1;
    while (j >= 0 && a[j] > v) {
        a[j + 1] = a[j];
        j--;
    }
    a[j + 1] = v;
}
```

#### ALDS1_1_B: Greatest Common Divisor

`while (b != 0)` で余りを回していくのが基本です。

#### ALDS1_1_C: Prime Numbers

`2` から `sqrt(n)` までの割り算確認で判定できます。

#### ALDS1_1_D: Maximum Profit

左から走査し、これまでの最小値を更新しながら最大利益を求めます。
