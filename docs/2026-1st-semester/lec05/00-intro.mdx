## 今回学ぶこと

### 1. ポインタとは

変数の**アドレス（メモリ上の番地）** を格納する変数です。

```c
int x = 10;
int *p = &x;  // p には x のアドレスが入る
printf("%d\n", *p);  // 10 と出力（間接参照）
```

### 2. 参照渡し

ポインタを引数に渡すことで、関数の外の変数を書き換えられます。

```c
void swap(int *a, int *b) {
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

int main() {
    int x = 3, y = 5;
    swap(&x, &y);
    printf("%d %d\n", x, y);  // 5 3 と出力
    return 0;
}
```

### 3. 動的メモリ確保

実行時にサイズが決まる配列を `malloc` で確保できます。

```c
#include <stdlib.h>

int n = 5;
int *arr = (int *)malloc(n * sizeof(int));
// ... arr を配列のように使う ...
free(arr);  // 必ず解放する
```
