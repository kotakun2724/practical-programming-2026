# ポインタの基礎

## ポインタとは

C言語では、すべての変数はメモリ上のどこかに格納されており、その場所を**アドレス**と呼びます。
**ポインタ**は、このアドレスを値として格納する特殊な変数です。

メモリ上のイメージ:

| アドレス | 値 | 変数名 |
|:---------|:---|:-------|
| 0x1000 | 10 | x |
| 0x1004 | --- | （未使用） |
| 0x1008 | 0x1000 | p (int *) |

## アドレス演算子 `&`

変数のアドレスを取得するには `&` を使います。

```c
int x = 10;
printf("%p\n", &x);  // x のアドレスを出力（例: 0x7ffd1234）
```

## ポインタの宣言

型名の後ろに `*` を付けて宣言します。

```c
int *p;    // int 型変数を指すポインタ
double *q; // double 型変数を指すポインタ
char *s;   // char 型変数を指すポインタ
```

## ポインタへの代入

```c
int x = 10;
int *p = &x;  // p に x のアドレスを代入
```

## 間接参照演算子 `*`

ポインタが指す先の値を読み書きするには `*` を使います。

```c
int x = 10;
int *p = &x;

printf("%d\n", *p);  // 10 と出力（x の値を読む）
*p = 20;             // x の値を 20 に書き換える
printf("%d\n", x);   // 20 と出力
```

:::warning `*` の2つの意味
- **宣言時** の `*`：ポインタ変数であることを示す（例: `int *p`）
- **式中** の `*`：ポインタが指す先の値を参照する（例: `*p`）
:::

## 参照渡し（ポインタを使った値渡し）

通常の関数引数は**値渡し**であり、関数内で引数を変更しても呼び出し元には影響しません。

```c
void add_one_val(int x) {
    x++;  // 呼び出し元の変数は変わらない
}
```

ポインタを引数に渡す**参照渡し**では、関数内から呼び出し元の変数を直接変更できます。

```c
void add_one_ptr(int *x) {
    (*x)++;  // 呼び出し元の変数を変更する
}

int main() {
    int a = 5;
    add_one_ptr(&a);     // アドレスを渡す
    printf("%d\n", a);   // 6 と出力
    return 0;
}
```

## swap 関数の例

参照渡しの代表例として、2つの変数の値を交換する `swap` 関数を見てみましょう。

```c
#include <stdio.h>

void swap(int *a, int *b) {
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

int main() {
    int x = 3, y = 5;
    printf("before: x=%d, y=%d\n", x, y);
    swap(&x, &y);
    printf("after:  x=%d, y=%d\n", x, y);
    return 0;
}
```

実行結果:
```
before: x=3, y=5
after:  x=5, y=3
```

## ポインタと配列

C言語では配列名はその先頭要素へのポインタとして扱われます。

```c
int a[5] = {10, 20, 30, 40, 50};
int *p = a;         // &a[0] と同じ

printf("%d\n", *p);       // 10 (a[0])
printf("%d\n", *(p + 1)); // 20 (a[1])
printf("%d\n", *(p + 2)); // 30 (a[2])
```

:::note 配列と関数
配列を関数に渡すとき、実際にはポインタが渡されます。
そのため、関数内で配列の中身を変更すると、呼び出し元の配列も変わります。

```c
void fill_zero(int *arr, int n) {
    for (int i = 0; i < n; i++) arr[i] = 0;
}
```
:::
