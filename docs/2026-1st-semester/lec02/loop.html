while文とfor文（繰り返し）

繰り返しとは​

同じ処理を何度も実行することを繰り返し（ループ）といいます。

例えば「Hello」を10回表示したい場合：

// これは大変...
printf("Hello\n");
printf("Hello\n");
printf("Hello\n");
// ...あと7回

繰り返し文を使えば簡潔に書けます：

for (int i = 0; i < 10; i++) {
    printf("Hello\n");
}

while文​

条件が真の間、処理を繰り返します。

while (条件) {
    // 条件が真の間、繰り返し実行
}

例：1から5まで表示​

#include <stdio.h>

int main(void) {
    int i = 1;
    
    while (i <= 5) {
        printf("%d\n", i);
        i++;  // iを1増やす（これがないと無限ループ！）
    }
    
    return 0;
}

出力：

1
2
3
4
5

:::danger 無限ループに注意 条件が永遠に真だと、プログラムが終わりません。

int i = 1;
while (i <= 5) {
    printf("%d\n", i);
    // i++を忘れると、iは1のままで条件が常に真
}

無限ループになったら Ctrl + C で強制終了できます。 :::

while文の流れ​

    ┌─────────────────┐
    │   条件をチェック   │
    └────────┬────────┘
             │
    ┌────────▼────────┐
    │   真（条件成立）   │──────┐
    └────────┬────────┘      │
             │               │
    ┌────────▼────────┐      │
    │   処理を実行      │      │
    └────────┬────────┘      │
             │               │
             └───────────────┘
             
    ┌────────▼────────┐
    │   偽（ループ終了）  │
    └─────────────────┘

for文​

回数が決まっている繰り返しに便利です。

for (初期化; 条件; 更新) {
    // 繰り返し実行
}
  • 初期化：ループ開始前に1回だけ実行
  • 条件：毎回チェック、偽になったらループ終了
  • 更新：毎回の繰り返しの最後に実行

例：1から5まで表示（for文版）​

#include <stdio.h>

int main(void) {
    for (int i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
    
    return 0;
}

for文の流れ​

for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}
  1. int i = 1 → iを1に初期化
  2. i <= 5 → 真なので処理実行、1を出力
  3. i++ → iが2になる
  4. i <= 5 → 真なので処理実行、2を出力
  5. i++ → iが3になる
  6. ...
  7. i++ → iが6になる
  8. i <= 5 → 偽なのでループ終了

whileとforの使い分け​

状況使う文
回数が決まっているfor文
回数が決まっていないwhile文
終了条件があるwhile文
// 10回繰り返す → for
for (int i = 0; i < 10; i++) { ... }

// 0が入力されるまで繰り返す → while
while (x != 0) {
    scanf("%d", &x);
}

AOJの問題を解いてみよう​

ITP1_3_A: Print Many Hello World​

「Hello World」を1000回出力する問題。

#include <stdio.h>

int main(void) {
    for (int i = 0; i < 1000; i++) {
        printf("Hello World\n");
    }
    
    return 0;
}

ITP1_3_B: Print Test Cases​

0が入力されるまで、ケース番号とともに値を出力する問題。

#include <stdio.h>

int main(void) {
    int x;
    int caseNum = 1;
    
    while (1) {  // 無限ループ
        scanf("%d", &x);
        if (x == 0) {
            break;  // ループを抜ける
        }
        printf("Case %d: %d\n", caseNum, x);
        caseNum++;
    }
    
    return 0;
}

ITP1_3_C: Swapping Two Numbers​

2つの数を小さい順に出力。0 0が入力されたら終了。

#include <stdio.h>

int main(void) {
    int x, y;
    
    while (1) {
        scanf("%d %d", &x, &y);
        if (x == 0 && y == 0) {
            break;
        }
        
        if (x > y) {
            // 交換
            int tmp = x;
            x = y;
            y = tmp;
        }
        
        printf("%d %d\n", x, y);
    }
    
    return 0;
}

break と continue​

break：ループを抜ける​

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // i==5でループ終了
    }
    printf("%d\n", i);
}
// 出力: 0, 1, 2, 3, 4

continue：次の繰り返しへスキップ​

for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;  // 偶数はスキップ
    }
    printf("%d\n", i);
}
// 出力: 1, 3, 5, 7, 9

二重ループ​

ループの中にループを書くことができます。

例：九九の表​

#include <stdio.h>

int main(void) {
    for (int i = 1; i <= 9; i++) {
        for (int j = 1; j <= 9; j++) {
            printf("%d ", i * j);
        }
        printf("\n");
    }
    
    return 0;
}

出力：

1 2 3 4 5 6 7 8 9 
2 4 6 8 10 12 14 16 18 
3 6 9 12 15 18 21 24 27 
...
