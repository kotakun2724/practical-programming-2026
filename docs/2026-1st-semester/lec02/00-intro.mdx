## 今回学ぶこと

### 1. 比較演算子・論理演算子

値を比較して **真（true）** か **偽（false）** を判定します。

```c
a == b  // aとbは等しい？
a < b   // aはbより小さい？
a && b  // aもbも真？
```

### 2. if文（条件分岐）

条件によって処理を分けます。

```c
if (x > 0) {
    printf("正の数です\n");
} else {
    printf("0以下です\n");
}
```

### 3. while文・for文（繰り返し）

同じ処理を何度も実行します。

```c
for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}
```

それでは始めましょう！
