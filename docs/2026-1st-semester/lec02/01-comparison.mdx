# 比較演算子と論理演算子

## 比較演算子

2つの値を比較して、**真（1）** または **偽（0）** を返します。

| 演算子 | 意味 | 例 | 結果 |
|:-------|:-----|:---|:-----|
| `==` | 等しい | `5 == 5` | 真（1） |
| `!=` | 等しくない | `5 != 3` | 真（1） |
| `<` | より小さい | `3 < 5` | 真（1） |
| `>` | より大きい | `5 > 3` | 真（1） |
| `<=` | 以下 | `5 <= 5` | 真（1） |
| `>=` | 以上 | `5 >= 3` | 真（1） |

:::warning `=` と `==` の違い
- `=` は**代入**（値を変数に入れる）
- `==` は**比較**（2つの値が等しいかチェック）

```c
int x = 5;    // 代入：xに5を入れる
if (x == 5)   // 比較：xが5と等しいか？
```

よくある間違い：`if (x = 5)` と書くと、比較ではなく代入になってしまう！
:::

### 比較の例

```c
#include <stdio.h>

int main(void) {
    int a = 10, b = 20;
    
    printf("a == b: %d\n", a == b);  // 0（偽）
    printf("a != b: %d\n", a != b);  // 1（真）
    printf("a < b: %d\n", a < b);    // 1（真）
    printf("a > b: %d\n", a > b);    // 0（偽）
    
    return 0;
}
```

## 論理演算子

複数の条件を組み合わせるときに使います。

| 演算子 | 意味 | 例 |
|:-------|:-----|:---|
| `&&` | AND（かつ） | `a > 0 && a < 10` |
| `\|\|` | OR（または） | `a < 0 \|\| a > 100` |
| `!` | NOT（否定） | `!(a == b)` |

### AND（&&）：両方とも真のとき真

```c
// aが0より大きく、かつ10より小さい
if (a > 0 && a < 10) {
    printf("aは1から9の間です\n");
}
```

| a > 0 | a &lt; 10 | a > 0 && a &lt; 10 |
|:------|:----------|:-------------------|
| 真 | 真 | **真** |
| 真 | 偽 | 偽 |
| 偽 | 真 | 偽 |
| 偽 | 偽 | 偽 |

### OR（||）：どちらかが真のとき真

```c
// aが0未満、または100より大きい
if (a < 0 || a > 100) {
    printf("aは範囲外です\n");
}
```

| a &lt; 0 | a > 100 | a &lt; 0 \|\| a > 100 |
|:---------|:--------|:----------------------|
| 真 | 真 | 真 |
| 真 | 偽 | **真** |
| 偽 | 真 | **真** |
| 偽 | 偽 | 偽 |

### NOT（!）：真偽を反転

```c
int finished = 0;  // 0は偽

if (!finished) {
    printf("まだ終わっていません\n");
}
```

## 実践例：範囲判定

AOJ ITP1_2_B「Range」のような問題を解くとき、論理演算子が役立ちます。

```c
#include <stdio.h>

int main(void) {
    int a, b, c;
    scanf("%d %d %d", &a, &b, &c);
    
    // aがbより小さく、かつbがcより小さいか？
    if (a < b && b < c) {
        printf("Yes\n");
    } else {
        printf("No\n");
    }
    
    return 0;
}
```

## 演算子の優先順位

算術演算子が先に評価され、その後比較、最後に論理演算子が評価されます。

```
高い  ()        括弧
 ↓   * / %     乗除
 ↓   + -       加減
 ↓   < > <= >= 比較
 ↓   == !=     等価
 ↓   &&        AND
低い  ||        OR
```

:::tip 迷ったら括弧を使う
優先順位が分からないときは、括弧 `()` で明示するのが安全です。

```c
// どちらも同じ意味だが、下の方が分かりやすい
if (a > 0 && b > 0 || c > 0)
if ((a > 0 && b > 0) || c > 0)
```
:::
