ITP1_2_A: Small, Large, or Equal​

今回習った if-else if-else をそのまま使えます。

ITP1_2_B: Range​

// ヒント：&&（AND）を使う
if (a < b && b < c) {
    printf("Yes\n");
}

ITP1_2_C: Sorting Three Numbers​

3つの値を比較して並べ替えます。講義資料のコードを参考に。

ITP1_2_D: Circle in a Rectangle​

// ヒント：円が長方形の内側に収まる条件
// 円の中心(cx, cy)、半径r
// 長方形の左下(x1, y1)、右上(x2, y2)

// 左端、右端、下端、上端すべてが範囲内か
if (cx - r >= x1 && cx + r <= x2 && 
    cy - r >= y1 && cy + r <= y2) {
    printf("Yes\n");
}

ITP1_3_A: Print Many Hello World​

for ループで1000回出力するだけ！

ITP1_3_B: Print Test Cases​

// ヒント：0が来るまで繰り返す
int x, caseNum = 1;
while (1) {
    scanf("%d", &x);
    if (x == 0) break;
    // 出力処理
    caseNum++;
}

ITP1_3_C: Swapping Two Numbers​

// ヒント：2つの値を交換する方法
int tmp = x;
x = y;
y = tmp;

ITP1_3_D: How Many Divisors?​

// ヒント：約数を数える
// a から b までの各整数について、c で割り切れるか調べる
for (int i = a; i <= b; i++) {
    if (c % i == 0) {
        count++;
    }
}

困ったときは​

よくあるエラー​

エラー原因対処法
CE (Compile Error)文法エラーセミコロン、括弧の対応を確認
WA (Wrong Answer)出力が間違っている出力形式を問題文と照らし合わせる

TLE (Time Limit Exceeded)時間超過ループ条件を確認（無限ループ？）
