特殊なパラメータを持つ RSA 暗号

RSA 暗号は、設計ルールを正しく守って実装しなければ安全性を保つことができません．

今回は、鍵生成時に使われる素数 p,qp, qp,q の選択が正しくない場合、数学的にどのように安全性が崩壊するのかを学びます．

1. 素数が1つだけの場合​

通常の RSA 暗号では、公開鍵 NNN は2つの異なる素数の積（N=pqN = pqN=pq）として生成され， φ(N)=(p−1)(q−1)\varphi(N) = (p-1)(q-1)φ(N)=(p−1)(q−1) を計算します．

しかし、NNN に素数が1つしか使われなかった場合（つまり N=pN = pN=p）はどうなるでしょうか？

秘密鍵を求める​

NNN が素数 ppp そのものである場合、オイラーの φ\varphiφ 関数の性質 より、以下が成り立ちます．

φ(N)=φ(p)=p−1=N−1\varphi(N) = \varphi(p) = p - 1 = N - 1φ(N)=φ(p)=p−1=N−1

これより，公開鍵 (e,N)(e, N)(e,N) から，秘密鍵 ddd を求めることができます．

d≡e−1(modN−1)d \equiv e^{-1} \pmod{N - 1}d≡e−1(modN−1)

秘密鍵 ddd が分かれば、通常通り暗号文を簡単に復号できます．

2. 素数が同一である場合​

次に、2つの素数 ppp と qqq をランダムに選んだつもりが、同じ素数を選んでしまった場合（つまり p=qp = qp=q ）を考えます． このとき、公開鍵 NNN は以下のように平方数になります．

N=p2N = p^2N=p2

RSA 暗号の安全性は「巨大な NNN の素因数分解が困難であること」に依存しています．しかし、 NNN がある整数の2乗（ N=p2N = p^2N=p2 ）である場合、素因数分解を効率的に行うことができます．

なぜなら、公開されている NNN に対して平方根を計算するだけでよいからです．

p=Np = \sqrt{N}p=N​

素因数分解を比較的効率的に行うアルゴリズムとして pollard's rho 法がありますが，これは指数時間のアルゴリズムです． それに反し，平方根を求める効率的なアルゴリズムであるニュートン法は，多項式時間アルゴリズムであるので，平方根を求める方が圧倒的に早く終わります．

秘密鍵を求める​

素因数 ppp が判明すれば、オイラーの φ\varphiφ 関数の性質 を利用して、 φ(N)\varphi(N)φ(N) を計算できます．

φ(N)=φ(p2)=p2−p=p(p−1)\varphi(N) = \varphi(p^2) = p^2 - p = p(p-1)φ(N)=φ(p2)=p2−p=p(p−1)

これより，公開鍵 (e,N)(e, N)(e,N) から秘密鍵 ddd を求めることができ，秘密鍵 ddd が求まれば，通常通り暗号文を復号できます．

このように，素因数分解の困難性に基づく暗号方式で，素因数分解を行う必要が無ければ，すぐに破ることができます．
