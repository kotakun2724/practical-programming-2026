### 問題のヒント

両問題とも `tar.gz` ファイルが配布されています．

`tar.gz` ファイルを手元にダウンロードし，

```
tar -zxvf one-p-rsa.tar.gz
```

を実行してください．

one-p-rsa であれば，`chall.py` と `chall.txt` が存在するはずです．

```chall.txt
p = 70530...
e = 65537
ct = 1207...
```

`chall.txt` には `p`, `e`, `ct` が記載されています．これは，`chall.py`　を実行した結果です．

`ct` は flag の暗号文であり，どういう手順で暗号化したのかは `chall.py` を見ることでわかります．

この渡された数値を用いて flag を取得しましょう．

<details>

<summary>ソルバーの雛形</summary>

```python
from Crypto.Util.number import long_to_bytes

# chall.txt で渡される数値
p = ...
e = ...
ct = ...

# int_my_flag を求めるためのコード
...

# 数値から文字列への変換
int_my_flag = ...

bytes_my_flag = long_to_bytes(int_my_flag)

my_flag = bytes_my_flag.decode('utf-8')

print(my_flag)
```

</details>
