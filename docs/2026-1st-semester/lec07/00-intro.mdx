## 今回学ぶこと

### 1. 再帰関数

関数が**自分自身を呼び出す**書き方です。ベースケース（停止条件）を必ず用意します。

```c
int factorial(int n) {
    if (n <= 1) return 1;        // ベースケース
    return n * factorial(n - 1); // 再帰呼び出し
}
```

### 2. 分割統治法

問題を**半分（あるいは複数）に分割**し、それぞれを解いた結果を**統合**して全体の答えを得る考え方です。

```
問題(N)
 ├─ 問題(N/2)  ─┐
 │              ├─ 統合して答えを作る
 └─ 問題(N/2)  ─┘
```

マージソートやクイックソートは、この考え方で `O(N log N)` を達成します。

### 3. スタック

最後に入れたものが最初に出てくる **LIFO（Last In First Out）** のデータ構造です。

```c
int stack[100];
int top = 0;

void push(int x) { stack[top++] = x; }
int  pop(void)   { return stack[--top]; }
```

関数の再帰呼び出しは、内部的にこの「スタック」（**コールスタック**）に積まれていきます。
