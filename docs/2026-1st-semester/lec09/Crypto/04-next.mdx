# 次回予告と課題

## 次回の内容

次回は，今回学んだ「拡張ユークリッドの互除法」を応用して，**逆元 (Modular Inverse)** について学びます．

## 今回の課題

### 1. Greatest Common Divisor
- カテゴリ: `Introduction` -> `Introduction to CryptoHack`
- 内容: 2つの非常に大きな数の最大公約数を求めます．
- ヒント: Python の `math.gcd` を使うか，自分で実装した関数を使ってみましょう．

### 2. Extended GCD
- カテゴリ: `Introduction` -> `Introduction to CryptoHack`
- 内容: $ax + by = \text{gcd}(a, b)$ を満たす整数 $x, y$ のうち，$x$ と $y$ のうち小さい方を答えます．（問題文をよく読んでください）
- ヒント: 先ほど学んだ `extended_gcd` 関数を実装して使ってみましょう．
