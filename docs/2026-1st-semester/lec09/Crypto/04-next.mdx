# 次回予告と課題

## 次回の内容

次回は，今回学んだ「拡張ユークリッドの互除法」を応用して，**逆元 (Modular Inverse)** について学びます．

逆元が理解できれば，いよいよ **RSA 暗号** の仕組みを紐解く準備が整います．お楽しみに！

## 今回の課題

学んだ知識を使って，実際に [CryptoHack](https://cryptohack.org/) の問題を解いてみましょう．

### CryptoHack の進め方

1. アカウントを作成する（無料です）．
2. [Courses / Challenges] から `Introduction to CryptoHack` を選択する．
3. 各問題のページにある "Download Source" や説明文を読み，コードを書く．
4. 答えが分かったら "Submit" 欄に入力する．

### 1. Greatest Common Divisor
- カテゴリ: `Introduction` -> `Introduction to CryptoHack`
- 内容: 2つの非常に大きな数の最大公約数を求めます．
- ヒント: Python の `math.gcd` を使うか，自分で実装した関数を使ってみましょう．

### 2. Extended GCD
- カテゴリ: `Introduction` -> `Introduction to CryptoHack`
- 内容: $ax + by = \text{gcd}(a, b)$ を満たす整数 $x, y$ のうち，$x$ と $y$ のうち小さい方を答えます．（問題文をよく読んでください）
- ヒント: 先ほど学んだ `extended_gcd` 関数を実装して使ってみましょう．

### 報告方法

フラグ（`crypto{...}` という形式の文字列）が手に入ったら，提出フォーム（LMS等）に記入してください．
