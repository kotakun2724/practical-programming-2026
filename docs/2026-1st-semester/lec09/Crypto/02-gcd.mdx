# 最大公約数 (GCD) とユークリッドの互除法

## 最大公約数 (Greatest Common Divisor) とは

2つ以上の整数の共通の約数のうち，最大のものを**最大公約数 (GCD)** と呼びます．

例えば，$12$ と $18$ の最大公約数は $6$ です．
- $12$ の約数: $\{1, 2, 3, 4, 6, 12\}$
- $18$ の約数: $\{1, 2, 3, 6, 9, 18\}$
- 共通の約数: $\{1, 2, 3, 6\}$ → 最大のものは $6$

## ユークリッドの互除法

大きな数の最大公約数を求める際，約数をすべて書き出すのは大変です．そこで使われるのが**ユークリッドの互除法**です．

### アルゴリズムの仕組み

2つの整数 $a, b$ ($a \ge b$) について，次のような性質があります．

> $a$ を $b$ で割った余りを $r$ とすると，$\text{gcd}(a, b) = \text{gcd}(b, r)$ である．

これを繰り返して，余りが $0$ になったときの割る数が最大公約数となります．

### 計算例: $\text{gcd}(111, 30)$

1. $111 \div 30 = 3$ 余り $21 \implies \text{gcd}(111, 30) = \text{gcd}(30, 21)$
2. $30 \div 21 = 1$ 余り $9 \implies \text{gcd}(30, 21) = \text{gcd}(21, 9)$
3. $21 \div 9 = 2$ 余り $3 \implies \text{gcd}(21, 9) = \text{gcd}(9, 3)$
4. $9 \div 3 = 3$ 余り $0 \implies \text{gcd}(9, 3) = 3$

よって，$\text{gcd}(111, 30) = 3$ です．

## Python での実装

Python では，再帰関数を使って簡潔に記述できます．

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

print(gcd(111, 30)) # 3
```

また，Python の標準ライブラリ `math` にも `gcd` 関数が用意されています．

```python
import math
print(math.gcd(111, 30)) # 3
```

CTFの暗号問題では，非常に大きな数を扱うため，自分でもこのアルゴリズムを理解しておくことが重要です．
