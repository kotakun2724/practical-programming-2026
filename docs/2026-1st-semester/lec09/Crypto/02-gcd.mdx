# 最大公約数 (GCD) とユークリッドの互除法

## 最大公約数 (Greatest Common Divisor)

2つ以上の整数の共通の約数のうち，最大のものを**最大公約数 (GCD)** と呼びます．

以降，2つ以上の整数 $a, b, \dots$ の最大公約数を $\gcd (a, b, \dots)$ を表記します．

### 例
- $\gcd (12, 18) = 6$
- $\gcd (12, 20) = 4$
- $\gcd (12, 7) = 1$

$\gcd (12, 7) = 1$ のように $\gcd (a, b) = 1$ であるとき，$a$ と $b$ は**互いに素**であるといいます．

## ユークリッドの互除法

$\gcd (225, 120) = 15$ です．このことが正しいかどうかを調べるために $225$ と $120$ を素因数分解し，両者の結果を比べるという方法があります．\
$255 = 3^{2} \cdot 5^{2}, \ 120 = 2^{3} \cdot 3 \cdot 5$ と素因数分解できるため，$\gcd (255, 120) = 15$ であることが確認できます．

大きな数の最大公約数を求める際に素因数分解して... は大変なので．**ユークリッドの互除法**がよく使われます．

### アルゴリズムの仕組み

$a, b$ を整数とします．$r_{-1} = a, r_{0} = b$ とおき，商と余りを求める以下の手続き

$$
r_{i-1} = q_{i+1} \times r_{i} + r_{i+1}
$$

を $i = 0, 1, 2, \dots$ について， $r_{n+1} = 0$ となるまで繰り返した時，$r_{n}$ は $a$ と $b$ の最大公約数となります．

まず，$r_{n}$ が $a, b$ の共通因数（公約数）であることを示します．

(6) より $r_{n}$ は $r_{n-1}$ の約数であることがわかります．

$r_{n}$ は $r_{n}$ と $r_{n-1}$ を割り切るため，$r_{n}$ は (5) の右辺を割り切ります．このことから，$r_{n}$ は $r_{n-2}$ の約数であることがわかります．

同様の議論を進めていくと，$r_{n}$ は $r_{2}$ と $r_{3}$ を割り切るため，$r_{1}$ の約数であるとわかります．

この時，$r_{n}$ は $b$ の約数であることもわかり，$b$ と $r_{1}$ も $r_{n}$ で割り切れるため，$a$ も $r_{n}$ で割り切れることがわかります．

よって，$r_{n}$ が $a, b$ の共通因数（公約数）であることがわかります．

次に，$r_{n}$ が公約数の中で最大であることを示します．

$a$ と $b$ の $r_{n}$ 以外の公約数を $d$ とします．

$d$ は $b$ と $r_{1}$ を割り切ります．このことから，$d$ は $r_{2}$ も割り切ります．

こうして議論を続けていくと，$d$ はどの $r_{i} \ (i = 1, \dots, n)$ も割り切ることがわかります．

(5) まで議論を続けていくと $d$ は $r_{n}$ を割り切ることがわかります．

つまり，$r_{n}$ が最大公約数でなければなりません．

### 計算例: $\text{gcd}(111, 30)$

$$
\begin{gather*}
    111 = 3 \times 30 + 21 \\
    30 = 1 \times 21 + 9 \\
    21 = 2 \times 9 + 3 \\
    9 = 3 \times 3 + 0
\end{gather*}
$$

よって，$\text{gcd}(111, 30) = 3$ です．

## Python での実装

ループ処理や再帰関数を用いて実装することができます．

```python
# while ループを用いた実装
def gcd(a, b):
    while b:
        tmp = a
        a = b
        b = tmp % b
    return a
```

また，Python の標準ライブラリ `math` にも `gcd` 関数が用意されています．

```python
import math
print(math.gcd(111, 30)) # 3
```
