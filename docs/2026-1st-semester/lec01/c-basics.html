C言語の基礎

AOJの問題を解くために必要なC言語の基礎を学びます。

プログラムの基本構造​

C言語のプログラムは以下の構造を持ちます。

#include <stdio.h>

int main(void) {
    // ここにプログラムを書く
    return 0;
}
要素説明
#include <stdio.h>入出力に必要なヘッダファイル
int main(void)プログラムの開始点
return 0;プログラムの正常終了

変数とデータ型​

変数とは、データを保存するための「箱」のようなものです。

変数の宣言​

int x;      // 整数を格納する変数 x を宣言
int a, b;   // 複数の変数を同時に宣言

変数の初期化​

int x = 10;     // 宣言と同時に値を代入
int a = 5, b = 3;

主なデータ型​

型説明例
int整数10, -5, 0
double小数（倍精度）3.14, -2.5
char1文字'A', 'x'

:::tip AOJでは主にint型を使用 ITP1の多くの問題では int 型で十分です。 :::

入出力​

出力：printf​

画面に文字や数値を表示します。

printf("Hello World\n");     // 文字列を出力
printf("%d\n", 10);          // 整数を出力
printf("%d %d\n", a, b);     // 複数の値を出力

フォーマット指定子​

指定子意味例
%d整数printf("%d", 42); → 42
%f小数printf("%f", 3.14); → 3.140000
%c1文字printf("%c", 'A'); → A
%s文字列printf("%s", "hello"); → hello

特殊文字​

文字意味
\n改行
\tタブ

入力：scanf​

キーボードから値を受け取ります。

int x;
scanf("%d", &x);    // 整数を入力して x に格納

:::warning 重要：&を忘れない scanf では変数名の前に &（アドレス演算子）が必要です。

scanf("%d", &x);    // ⭕ 正しい
scanf("%d", x);     // ❌ 間違い（&がない）

:::

複数の値を入力​

int a, b;
scanf("%d %d", &a, &b);    // スペース区切りで2つの整数を入力

算術演算子​

演算子意味例
+加算a + b
-減算a - b
*乗算a * b
/除算a / b
%剰余（余り）a % b

整数同士の除算に注意​

int a = 7, b = 2;
printf("%d\n", a / b);    // 結果: 3（小数点以下は切り捨て）
printf("%d\n", a % b);    // 結果: 1（余り）

ITP1_1の問題に挑戦​

ITP1_1_A: Hello World​

「Hello World」と出力するだけの問題です。

#include <stdio.h>

int main(void) {
    printf("Hello World\n");
    return 0;
}

ITP1_1_B: X Cubic​

整数 x を読み込んで、x の3乗（x³）を出力します。

#include <stdio.h>

int main(void) {
    int x;
    scanf("%d", &x);
    printf("%d\n", x * x * x);
    return 0;
}

ITP1_1_C: Rectangle​

長方形の縦 a と横 b を読み込んで、面積と周の長さを出力します。

#include <stdio.h>

int main(void) {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d %d\n", a * b, 2 * (a + b));
    return 0;
}

ITP1_1_D: Watch​

秒数 S を読み込んで、h:m:s の形式で出力します。

#include <stdio.h>

int main(void) {
    int S;
    scanf("%d", &S);
    
    int h = S / 3600;           // 時間
    int m = (S % 3600) / 60;    // 分
    int s = S % 60;             // 秒
    
    printf("%d:%d:%d\n", h, m, s);
    return 0;
}

:::tip 剰余演算子の活用 時間の変換では %（剰余）がよく使われます。

  • S / 3600 : 秒を3600で割ると時間が求まる
  • S % 3600 : 3600で割った余りから分と秒を計算
  • S % 60 : 60で割った余りが秒 :::
