# はじめてのCプログラム

いよいよC言語のプログラムを書いて実行してみましょう！

## Hello, World!

プログラミングの伝統として、最初に書くプログラムは「Hello, World!」と表示するものです。

### 1. ファイルの作成

ターミナルで作業ディレクトリに移動し、ファイルを作成します。

```bash
cd ~/programming/c-lang
touch hello.c
```

### 2. プログラムを書く

VSCodeで `hello.c` を開き、以下のコードを入力してください。

```c title="hello.c"
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}
```

### 3. コンパイル

C言語のプログラムは、そのままでは実行できません。
**コンパイラ**を使って**機械語**（コンピュータが理解できる言語）に変換する必要があります。

```bash
gcc hello.c -o hello
```

このコマンドの意味：
- `gcc` : Cコンパイラ
- `hello.c` : コンパイルするソースファイル
- `-o hello` : 出力ファイル名を `hello` にする

### 4. 実行

```bash
./hello
```

出力：
```
Hello, World!
```

🎉 **おめでとうございます！** 最初のCプログラムが実行できました！

## コードの解説

```c
#include <stdio.h>
```
**ヘッダファイルのインクルード**
- `stdio.h` は「Standard Input/Output」の略
- `printf` 関数を使うために必要

```c
int main(void) {
```
**main関数の定義**
- プログラムは必ず `main` 関数から実行される
- `int` は戻り値の型（整数）
- `void` は引数がないことを示す

```c
    printf("Hello, World!\n");
```
**画面への出力**
- `printf` は文字列を画面に表示する関数
- `\n` は改行を意味する（エスケープシーケンス）

```c
    return 0;
```
**プログラムの終了**
- `0` を返すことで「正常終了」を示す

## 練習問題

### 練習1: 自己紹介プログラム

自分の名前を表示するプログラムを作成してください。

```bash
touch intro.c
```

```c title="intro.c"
#include <stdio.h>

int main(void) {
    printf("私の名前は〇〇です。\n");
    printf("趣味は△△です。\n");
    return 0;
}
```

コンパイルと実行：
```bash
gcc intro.c -o intro
./intro
```

### 練習2: 複数行の出力

以下のような出力を作成してください。

```
********************
*                  *
*   Hello, C!      *
*                  *
********************
```

<details>
<summary>ヒント</summary>

`printf` を複数回使うか、`\n` で改行を入れて1つの `printf` で書くこともできます。

</details>

<details>
<summary>解答例</summary>

```c title="box.c"
#include <stdio.h>

int main(void) {
    printf("********************\n");
    printf("*                  *\n");
    printf("*   Hello, C!      *\n");
    printf("*                  *\n");
    printf("********************\n");
    return 0;
}
```

</details>

### 練習3: エスケープシーケンス

以下のエスケープシーケンスを使って、色々な出力を試してみましょう。

| エスケープシーケンス | 意味 |
|---------------------|------|
| `\n` | 改行 |
| `\t` | タブ（水平） |
| `\\` | バックスラッシュ |
| `\"` | ダブルクォート |
| `\'` | シングルクォート |

```c title="escape.c"
#include <stdio.h>

int main(void) {
    printf("名前\t年齢\t趣味\n");
    printf("田中\t20\tプログラミング\n");
    printf("鈴木\t21\t読書\n");
    printf("\n");
    printf("パス: C:\\Users\\tanaka\n");
    printf("彼は\"すごい\"と言った。\n");
    return 0;
}
```

## よくあるエラーと対処法

### コンパイルエラー

```
hello.c:4:5: error: expected ';' after expression
    printf("Hello, World!\n")
                             ^
                             ;
```

**原因**: 文の最後のセミコロン `;` が抜けている

**対処**: エラーメッセージの指摘箇所を確認し、`;` を追加

### リンクエラー

```
undefined reference to `main'
```

**原因**: `main` 関数が存在しない、またはスペルミス

**対処**: `main` 関数が正しく定義されているか確認

### 実行時エラー

```
bash: ./hello: No such file or directory
```

**原因**: コンパイルしていない、または違うディレクトリにいる

**対処**: `ls` でファイルを確認し、`gcc` でコンパイル
