# 次回予告と課題

## 次回の内容

**第2回: C言語入門 - 変数と型**

次回は以下の内容を学びます：

- 変数の宣言と初期化
- データ型（`int`, `float`, `double`, `char`）
- 算術演算子（`+`, `-`, `*`, `/`, `%`）
- `scanf` を使った入力
- 簡単な計算プログラム

```c title="次回の予告コード"
#include <stdio.h>

int main(void) {
    int age;
    float height;
    
    printf("年齢を入力してください: ");
    scanf("%d", &age);
    
    printf("身長(cm)を入力してください: ");
    scanf("%f", &height);
    
    printf("あなたは %d 歳で、身長は %.1f cm です。\n", age, height);
    
    return 0;
}
```

## 質問の仕方

プログラミングで困ったときは、以下の情報を含めて質問してください：

### 良い質問の例

```
【やりたいこと】
Hello, World! と表示したい

【試したこと】
gcc hello.c -o hello を実行した

【エラーメッセージ】
hello.c:4:5: error: expected ';' after expression

【ソースコード】
（hello.c の内容を貼り付け）

【環境】
macOS, gcc 15.0.0
```

### 質問するときのポイント

1. **何をしようとしたか**を明確に
2. **何を試したか**を書く
3. **エラーメッセージ**は省略せずにコピペ
4. **ソースコード**を共有する
5. **環境**（OS、コンパイラのバージョン）を書く

:::warning やってはいけない質問
❌「動きません」（何が？どう動かない？）
❌「エラーが出ます」（どんなエラー？）
❌「教えてください」（何を？）

具体的な情報がないと、助けることができません。
:::
