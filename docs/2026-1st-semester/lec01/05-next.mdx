### 各問題のヒント

#### ITP1_1_A: Hello World

今回の講義で学んだ `printf` を使えばOKです。  
「C言語の基本」ページのサンプルコードを参考にしてください。

#### ITP1_1_B: X Cubic（立方体）

```c
// ヒント：3乗の計算
int x;
scanf("%d", &x);
printf("%d\n", x * x * x);
```

#### ITP1_1_C: Rectangle（長方形）

```c
// ヒント：2つの値を読み込む
int a, b;
scanf("%d %d", &a, &b);
// 面積 = a * b
// 周長 = 2 * (a + b)
```

#### ITP1_1_D: Watch（時計）

```c
// ヒント：割り算と余りを使う
int S;
scanf("%d", &S);
int h = S / 3600;        // 時間
int m = (S % 3600) / 60; // 分
int s = S % 60;          // 秒
printf("%d:%d:%d\n", h, m, s);
```

:::tip 出力形式に注意
AOJは出力形式に厳密です。余計な空白や文字があるとWA（Wrong Answer）になります。  
問題文のOutput欄をよく読んで、フォーマットを確認しましょう。
:::

## 困ったときは

### よくあるエラー

| エラー | 原因 | 対処法 |
|:-------|:-----|:-------|
| CE (Compile Error) | 文法エラー | セミコロン、括弧の対応を確認 |
| WA (Wrong Answer) | 出力が間違っている | 出力形式を問題文と照らし合わせる |
| RE (Runtime Error) | 実行時エラー | 0除算、配列の範囲外アクセスなど |
| TLE (Time Limit Exceeded) | 時間超過 | 今回の問題では起きにくい |
