# C言語の基礎

AOJの問題を解くために必要なC言語の基礎を学びます。

## プログラムの基本構造

C言語のプログラムは以下の構造を持ちます。

```c
#include <stdio.h>

int main(void) {
    // ここにプログラムを書く
    return 0;
}
```

| 要素 | 説明 |
|------|------|
| `#include <stdio.h>` | 入出力に必要なヘッダファイル |
| `int main(void)` | プログラムの開始点 |
| `return 0;` | プログラムの正常終了 |

## 変数とデータ型

**変数**とは、データを保存するための「箱」のようなものです。

### 変数の宣言

```c
int x;      // 整数を格納する変数 x を宣言
int a, b;   // 複数の変数を同時に宣言
```

### 変数の初期化

```c
int x = 10;     // 宣言と同時に値を代入
int a = 5, b = 3;
```

### 主なデータ型

| 型 | 説明 | 例 |
|----|------|-----|
| `int` | 整数 | `10`, `-5`, `0` |
| `double` | 小数（倍精度） | `3.14`, `-2.5` |
| `char` | 1文字 | `'A'`, `'x'` |

:::tip AOJでは主にint型を使用
ITP1の多くの問題では `int` 型で十分です。
:::

## 入出力

### 出力：printf

画面に文字や数値を表示します。

```c
printf("Hello World\n");     // 文字列を出力
printf("%d\n", 10);          // 整数を出力
printf("%d %d\n", a, b);     // 複数の値を出力
```

#### フォーマット指定子

| 指定子 | 意味 | 例 |
|--------|------|-----|
| `%d` | 整数 | `printf("%d", 42);` → `42` |
| `%f` | 小数 | `printf("%f", 3.14);` → `3.140000` |
| `%c` | 1文字 | `printf("%c", 'A');` → `A` |
| `%s` | 文字列 | `printf("%s", "hello");` → `hello` |

#### 特殊文字

| 文字 | 意味 |
|------|------|
| `\n` | 改行 |
| `\t` | タブ |

### 入力：scanf

キーボードから値を受け取ります。

```c
int x;
scanf("%d", &x);    // 整数を入力して x に格納
```

:::warning 重要：&を忘れない
`scanf` では変数名の前に `&`（アドレス演算子）が必要です。
```c
scanf("%d", &x);    // ⭕ 正しい
scanf("%d", x);     // ❌ 間違い（&がない）
```
:::

#### 複数の値を入力

```c
int a, b;
scanf("%d %d", &a, &b);    // スペース区切りで2つの整数を入力
```

## 算術演算子

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `+` | 加算 | `a + b` |
| `-` | 減算 | `a - b` |
| `*` | 乗算 | `a * b` |
| `/` | 除算 | `a / b` |
| `%` | 剰余（余り） | `a % b` |

### 整数同士の除算に注意

```c
int a = 7, b = 2;
printf("%d\n", a / b);    // 結果: 3（小数点以下は切り捨て）
printf("%d\n", a % b);    // 結果: 1（余り）
```

## ITP1_1の問題に挑戦

### ITP1_1_A: Hello World

「Hello World」と出力するだけの問題です。

```c
#include <stdio.h>

int main(void) {
    printf("Hello World\n");
    return 0;
}
```

### ITP1_1_B: X Cubic

整数 x を読み込んで、x の3乗（x³）を出力します。

```c
#include <stdio.h>

int main(void) {
    int x;
    scanf("%d", &x);
    printf("%d\n", x * x * x);
    return 0;
}
```

### ITP1_1_C: Rectangle

長方形の縦 a と横 b を読み込んで、面積と周の長さを出力します。

```c
#include <stdio.h>

int main(void) {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d %d\n", a * b, 2 * (a + b));
    return 0;
}
```

### ITP1_1_D: Watch

秒数 S を読み込んで、`h:m:s` の形式で出力します。

```c
#include <stdio.h>

int main(void) {
    int S;
    scanf("%d", &S);
    
    int h = S / 3600;           // 時間
    int m = (S % 3600) / 60;    // 分
    int s = S % 60;             // 秒
    
    printf("%d:%d:%d\n", h, m, s);
    return 0;
}
```

:::tip 剰余演算子の活用
時間の変換では `%`（剰余）がよく使われます。
- `S / 3600` : 秒を3600で割ると時間が求まる
- `S % 3600` : 3600で割った余りから分と秒を計算
- `S % 60` : 60で割った余りが秒
:::
