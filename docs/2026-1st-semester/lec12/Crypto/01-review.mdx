# 前回課題解説

## Euler's Totient

> Given $N = p \cdot q$ and two primes:
> $$
> \begin{align*}
>   &p = 857504083339712752489993810777 \\
>   &q = 1029224947942998075080348647219
> \end{align*}
> $$
> What is Euler's totient $\phi(N)$?

$$
\begin{align*}
  &p = 857504083339712752489993810777 \\
  &q = 1029224947942998075080348647219
\end{align*}
$$

上記，$p, q$ の時，$\varphi(N = p \cdot q)$ を求める問題でした．

../../lec11/Crypto

$$
\varphi(N) = (p-1)(q-1)
$$

### 解法コード例

Python を使って以下のように計算できます．

```python
p = 101
q = 1009
phi = (p - 1) * (q - 1)
print(phi)
```

```shell
$ python solve.py
100800
```

## Private Keys

> $p = 857504081257975215582376885387731734091$
> $q = 102922494794299807508034864721971717161$
> $e = 65537$
> 
> Find the private key $d$.

RSA 暗号における秘密鍵（復号指数） $d$ を求める問題です．

RSA 暗号では，公開鍵 $e$ と秘密鍵 $d$ の間には以下の関係が成り立ちます．

$$
ed \equiv 1 \pmod{\varphi(N)}
$$

つまり， $d$ は法 $\varphi(N)$ における $e$ の**モジュラー逆数**です．
したがって，前々回（第10回）で学んだモジュラー逆数の求め方を使って $d$ を計算します．

### 解法コード例

Python 3.8 以降であれば，組み込み関数 `pow(base, exp, mod)` で `exp = -1` を指定することで，高速にモジュラー逆数を求めることができます．

```python
p = 857504081257975215582376885387731734091
q = 102922494794299807508034864721971717161
e = 65537

# N と phi の計算
N = p * q
phi = (p - 1) * (q - 1)

# e の mod phi における逆元 d を求める
d = pow(e, -1, phi)
print(d)
```

```shell
$ python solve.py
12188448554284898150493356066270513904940250917228859732386927508103099908121
```

手動で拡張ユークリッドの互除法を実装して求めることも可能ですが，Python の `pow` を使うのが最も簡潔です．
