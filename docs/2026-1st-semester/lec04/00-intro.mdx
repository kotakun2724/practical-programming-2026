## 今回学ぶこと

### 1. 関数の宣言と定義

複数の処理を関数として切り出します。

```c
int add(int a, int b) {
    return a + b;
}
```

### 2. 引数と戻り値

関数に値を渡して、結果を返すことができます。

```c
int result = add(3, 5);  // result は 8
```

### 3. スコープ

関数内で宣言した変数は、その関数内でのみ有効です。

```c
void func() {
    int x = 10;
    // x はこの関数内でのみ有効
}
// ここから x にアクセスするとエラー
```

### 4. 関数で実装するパターン

同じ処理を何度も書く代わりに、関数として実装します。

```c
// 1. 矩形を描画する関数
void print_rectangle(int h, int w) {
    for (int i = 0; i < h; i++) {
        for (int j = 0; j < w; j++) {
            printf("#");
        }
        printf("\n");
    }
}

// 2. 枠を描画する関数
void print_frame(int h, int w) {
    // ...
}
```
