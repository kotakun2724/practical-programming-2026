### 各問題のヒント

#### 1. Euler's Totient

二つの素数 $p, q$ が与えられるので，その合成数 $N = pq$ のオイラーの $\varphi$ 関数 $\varphi(N)$ を求めるだけの問題です．

#### 2. Private Keys

これは，次回内容の予習のような立ち位置です．

問題内に出てくる $d$ は復号時に重要な値になります．その $d$ は 法 $\varphi(N)$ で $e$ のモジュラー逆数を求めることで得られます．その $d$ を求める問題です．

## 困ったときは

### よくあるエラー

| エラー | 原因 | 対処法 |
|:-------|:-----|:-------|
| `RecursionError` | 再帰が深すぎる | 非常に大きな数を扱う場合、ループによる実装を検討するか `sys.setrecursionlimit` を調整する（今回の課題規模なら再帰で問題ありません） |
| `TypeError` | 演算の型の不一致 | Python 3 の `/` は小数になります。整数同士の割り算（商）を求めたい場合は `//` を使いましょう。 |
| Flag が通らない | 出力形式ミス | $x$ と $y$ のうち「小さい方」を答えているか再確認してください。負の数の方が小さいことに注意。 |
