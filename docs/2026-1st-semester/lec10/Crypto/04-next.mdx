## 困ったときは

### よくあるエラー

| エラー | 原因 | 対処法 |
|:-------|:-----|:-------|
| `RecursionError` | 再帰が深すぎる | 非常に大きな数を扱う場合、ループによる実装を検討するか `sys.setrecursionlimit` を調整する（今回の課題規模なら再帰で問題ありません） |
| `TypeError` | 演算の型の不一致 | Python 3 の `/` は小数になります。整数同士の割り算（商）を求めたい場合は `//` を使いましょう。 |
| Flag が通らない | 出力形式ミス | $x$ と $y$ のうち「小さい方」を答えているか再確認してください。負の数の方が小さいことに注意。 |
