# 配列の基礎

## 配列とは

配列は、**同じ型のデータを連続して保存**する仕組みです。

```c
int score[5];
```

この場合、`score[0]` から `score[4]` までの5個の整数を扱えます。

## 添字（インデックス）

C言語の配列は **0から始まる** ことに注意しましょう。

| 要素 | 添字 |
|:-----|:-----|
| 1個目 | `0` |
| 2個目 | `1` |
| 3個目 | `2` |

## 配列とfor文

```c
#include <stdio.h>

int main(void) {
    int a[5];

    for (int i = 0; i < 5; i++) {
        scanf("%d", &a[i]);
    }

    for (int i = 0; i < 5; i++) {
        printf("a[%d] = %d\n", i, a[i]);
    }

    return 0;
}
```

## 合計値を求める例

```c
#include <stdio.h>

int main(void) {
    int n;
    int a[100];
    int sum = 0;

    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
        sum += a[i];
    }

    printf("%d\n", sum);
    return 0;
}
```

:::warning 配列の範囲外アクセス
`a[100]` を宣言した場合、有効な添字は `0` から `99` です。
範囲外アクセスはバグや実行時エラーの原因になります。
:::
