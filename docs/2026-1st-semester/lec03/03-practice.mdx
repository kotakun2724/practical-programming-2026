## ヒント

### ITP1_4_D: Min, Max and Sum

```c
long long sum = 0;
int minv, maxv;

for (int i = 0; i < n; i++) {
    scanf("%d", &a[i]);
    if (i == 0) {
        minv = maxv = a[i];
    } else {
        if (a[i] < minv) minv = a[i];
        if (a[i] > maxv) maxv = a[i];
    }
    sum += a[i];
}
```

:::tip 出力形式を最後に再確認
AOJではスペース・改行位置まで採点対象です。
提出前にサンプル入出力と完全一致しているか確認しましょう。
:::
