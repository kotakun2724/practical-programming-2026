第3回: 配列と文字列

今回の目標​

  1. 配列を使って複数データをまとめて扱えるようになる
  2. 配列とfor文を組み合わせて処理を書けるようになる
  3. **文字列（char配列）**の基本を理解する
  4. strlen、strcmp などの文字列関数を使えるようになる

前回の復習​

前回は以下を学びました：

  • 比較演算子・論理演算子（==, <, && など）
  • if文 / else if文による条件分岐
  • while文 / for文による繰り返し
  • AOJ ITP1_2, ITP1_3 の基本問題

前回課題（ITP1_2A〜D, ITP1_3A〜D）の解説は、 次のページにまとめています。

  • 前回課題解説（C/C++）

今回学ぶこと​

1. 配列の宣言と利用​

同じ型の値をまとめて管理できます。

int a[5];

2. 配列とループ​

for (int i = 0; i < 5; i++) {
    scanf("%d", &a[i]);
}

3. 文字列（char配列）​

char s[101];
scanf("%100s", s);

4. 文字列関数​

strlen, strcmp を使って長さや比較ができます。

それでは始めましょう！
