## 今回学ぶこと

### 1. 配列の宣言と利用

同じ型の値をまとめて管理できます。

```c
int a[5];
```

### 2. 配列とループ

```c
for (int i = 0; i < 5; i++) {
    scanf("%d", &a[i]);
}
```

### 3. 文字列（char配列）

```c
char s[101];
scanf("%100s", s);
```

### 4. 文字列関数

`strlen`, `strcmp` を使って長さや比較ができます。

それでは始めましょう！
