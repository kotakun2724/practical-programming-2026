# 2026年度 実践的プログラミング

プログラミングの基礎から応用まで、実践的に学ぶ講義です。

## 講義概要

この講義では、**プログラミングの基礎力**と**実践的な開発スキル**を身につけることを目標としています。

### 前期：C/C++とアルゴリズム

**C/C++プログラミング言語と基本的なアルゴリズムの習得**を目的とし、多くの問題をステップbyステップで解いていきます。

- C言語の基礎文法
- データ構造（配列、構造体、ポインタ）
- 基本的なアルゴリズム
- メモリ管理

### 後期：Web開発

**Java, JavaScript, その他WEB技術を用いて、WEBサービスやアプリケーションの開発**を行います。

- HTML/CSS/JavaScript
- フロントエンド開発
- バックエンド開発
- データベース連携
