# 到達目標

この講義を通じて習得する技術とスキルについて説明します。

## 前期の到達目標

前期終了時に、以下のことができるようになることを目指します。

### 1. C言語の基礎をマスターする

```c
// こんなプログラムが書けるようになります
#include <stdio.h>

int main() {
    int numbers[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(numbers) / sizeof(numbers[0]);
    
    // バブルソートで並べ替え
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (numbers[j] > numbers[j + 1]) {
                int temp = numbers[j];
                numbers[j] = numbers[j + 1];
                numbers[j + 1] = temp;
            }
        }
    }
    
    printf("ソート結果: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", numbers[i]);
    }
    return 0;
}
```

### 2. 基本的なアルゴリズムを理解し実装できる

| アルゴリズム | 内容 |
|-------------|------|
| 線形探索 | 配列を順番に調べる |
| 二分探索 | ソート済み配列を効率的に探索 |
| バブルソート | 隣接要素を比較して並べ替え |
| 選択ソート | 最小値を選んで並べ替え |
| 挿入ソート | 適切な位置に挿入して並べ替え |

### 3. ポインタとメモリ管理を理解する

```c
// ポインタの基本
int value = 42;
int *ptr = &value;  // valueのアドレスを格納
printf("値: %d, アドレス: %p\n", *ptr, ptr);

// 動的メモリ確保
int *array = (int *)malloc(10 * sizeof(int));
// ... 使用後は解放
free(array);
```

### 4. 構造体を使ったデータ管理ができる

```c
typedef struct {
    char name[50];
    int age;
    float gpa;
} Student;

Student s1 = {"田中太郎", 20, 3.5};
```
